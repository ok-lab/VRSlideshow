# VR Slide show

Unity + Photon でVRデバイスを用いたオンラインスライドショーを実現する

各種スクリプトを残す

# Requirement

* Unity 
* PUN2 (Photon for Unity)
* PUN Voice2

# Installation

UnityのAssets StoreからPUN2もしくは、PUN Voice2をDLしてimportしてください
